Бинарный файл сразу же встречает нас открытием двух файлов `flag.txt` и `rev_this` с помощью функции `_fopen` профиль которой выглядит следующим 
образом: 
```c
FILE *fopen(const char *restrict pathname, const char *restrict mode);
```

В качестве параметров ей передается имя файла и режим с помощью которых мы открываем данные файлы. Вот некоторые из них: 
| Режим | Значение |
| :----- | :------- |
